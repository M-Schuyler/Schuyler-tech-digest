# WeChat Mini Program Wrapper - Reaction Speed Test

This mini program opens the reaction speed test H5 page via `web-view`.

## What to edit

- `project.config.json`: replace `wxYOUR_APPID` with your Mini Program AppID.
- `pages/reaction/index.js`: replace the `url` with your deployed H5 URL.

## Run

1. Deploy the H5 page from `../reaction-speed-test/index.html` to a public domain.
2. Add that domain to the Mini Program `web-view` business domain list and upload the verification file.
3. Open this folder in WeChat DevTools and run.
