# WeChat Mini Program - Reaction Speed Test

Native mini program page (no `web-view`, no business domain needed).

## What to edit

- `project.config.json`: replace `wxYOUR_APPID` with your Mini Program AppID.

## Run

1. Open this folder in WeChat DevTools.
2. Compile and preview.

## Notes

- Best score is stored via local storage key `reaction-best-ms`.
